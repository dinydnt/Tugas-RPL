ArgoUML adalah sebuah open source UML modelling tool yang dapat membuat berbagai macam diagram UML yang standard

ArgoUML merupakan software UML modeling tool open source dan menyediakan dukungan untuk semua standar diagram UML (Unified Modeling Language). Kode-kode program ArgoUML ditulis sepenuhnya dengan menggunakan bahasa pemrograman Java dan menggunakan Java Foundation Classess. Selain itu, ArgoUML juga kompatibel dengan standar OMG untuk UML 1.4. Software UML modeling tool ini berjalan pada semua platform Java dan tersedia dalam berbagai bahasa (sekitar 10 bahasa).

Computer Aided Software Engineering (CASE) mengotomatisasikan proses pembuatan software dari depan ke belakang. Sedangkan Unified Modeling Language (UML) menyediakan interoperabilitas (interoperability) yang memungkinkan mendapat manfaat dari CASE. ArgoUML merupakan suatu UML CASE tool untuk menganalisa dan merancang software berorientasi objek (object oriented software). Selain merupakan software yang bersifat open source, ArgoUML memiliki banyak aspek unik yang membedakannya dari tool-tool komersial, termasuk desain yang dapat meningkatkan produktivitas programmer serta dikembangkan pada sifat alami bahasa pemrograman Java yang memungkinkan akses dan kompatibilitas yang luas.

Installer ArgoUML memungkinkan Anda untuk menginstal Java Runtime Environment, apabila komputer Anda tidak memilikinya. Jika Anda telah memiliki versi Java Runtime Environment terbaru, maka Anda dapat melewat tahapan tersebut. Setelah software ArgoUML terinstal, maka Anda akan mendapati suatu desain antar muka alias interface yang begitu memudahkan, dirancang dengan baik, memiliki antar muka yang dapat Anda sesuaikan kembali (customizable interface), dan berbagai fitur lainnya yang membuat pekerjaan membuat diagram UML menjadi lebih mudah dan efisien.

Beberapa tampilan seperti resizable panel, graphic-style toolkit, dan detail tampilan lainnya terbilang cukup umum pada UML modeling tool saat ini. Namun ArgoUML memiliki beberapa sentuhan unik seperti panel prioritas (work prioritizing panel) yang memungkinkan Anda untuk menyesuaikan prioritas kerja sesuai dengan prioritas yang paling tinggi. Walau begitu banyak hal untuk dipelajari pada ArgoUML, penjelasan dan panduan wizard yang sangat baik membuatnya dapat diakses oleh berbagai tingkat pengguna. Selain itu, dengan bantuan file Help, Anda tidak akan merasa kesulitan ketika ingin menggunakan berbagai fungsionalitas yang terdapat di dalam software ini. ArgoUML menyediakan semua fitur-fiturnya secara gratis, oleh karena itu Anda patut mencobanya.
